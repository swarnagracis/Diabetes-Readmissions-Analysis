**Diabetes Readmission Analysis Dashboard**

**Project Overview**

This project presents a comprehensive analysis of a diabetic hospital encounters dataset using Power BI.

The objective was to explore:

-30-Day hospital readmissions

-Clinical risk factors

-Operational utilization patterns

-Resource consumption trends

-Admission & discharge performance

The final output is an interactive Power BI dashboard structured into three analytical layers:

1.Executive Overview

2.Clinical Insights & Risk Factors

3.Utilization & Operational Efficiency

**Dataset Summary**

-Total Encounters: ~99,000

-Average Length of Stay: 4.38 days 

-Average Medications per Encounter: 15.98 

-Overall Readmission Rate: 47% 

-30-Day Readmission Rate (Primary Diabetes): 13% 

The dataset includes:

-Demographics (Age, Gender, Race)

-Diagnoses (Primary & Secondary)

-Admission and discharge types

-Medication data

-Visit history (Inpatient, Outpatient, Emergency)

-Laboratory utilization

-Readmission flags
